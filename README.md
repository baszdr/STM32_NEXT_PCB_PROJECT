# STM32_NEXT_PCB_PROJECT
Designed a compact embedded system integrating USB-C, power management, and RF design into a single 4-layer PCB. Focused on clean layout, strong signal integrity, and practical debugging structure. 
💡 Features: 
• STM32WB55 (BLE)
• USB-C with protection
• 3.3V regulation 
• RF matching + U.FL • SWD debugging
